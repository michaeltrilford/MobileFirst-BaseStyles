Base: Font Size

Even though the CSS is structured in a Mobile First approach, the desktop font-size of 16px is used to calulate the rem units for devices below desktop.

Small Devices:

12px / 16px = 0.75rem

Medium Devices:

14px / 16px = 0.875rem

Large Devices:

16px / 16px = 1rem 
                
    html {
      font-size: 0.75rem;
    }
    @media only screen and (min-width: 48rem)
      html {
        font-size: 0.875rem;
      }
    @media only screen and (min-width: 53.125rem)
      html {
        font-size: 1rem;
      }

Base: Line Height

Small Devices:

1.5 * 12px = 18px

Medium Devices:

1.5 * 14px = 21px

Large Devices:

1.5 * 16px = 24px 
                
    html {
      line-height: 1.5;
    }
